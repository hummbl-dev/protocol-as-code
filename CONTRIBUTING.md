# Contributing

This is an early public seed repository. Contributions should keep the repository docs-first, auditable, and easy to review.

## Guidelines

- Keep examples small and inspectable.
- Do not add secrets, credentials, private operational details, or private receipts.
- Mark speculative material as speculative.
- Separate general patterns from HUMMBL/BaseN/Ownward-specific candidates.
- Prefer issues for scope questions before large changes.
